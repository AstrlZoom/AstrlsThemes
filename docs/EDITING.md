# AstrlGlass Editing Guide

This project is set up so most changes happen in one file:

`themes/AstrlGlass.theme.css`

Start there when you want to change colors, transparency, the DMs icon bounce, or the visible theme metadata.

## What To Edit

- `themes/AstrlGlass.theme.css`: easy settings, theme name, colors, on/off switches.
- `src/dms-button.css`: custom DMs/home icon shape, bounce animation, selected-state fixes.
- `src/desktop-transparency.css`: see-through Discord window support.
- `scripts/theme.config.js`: file order for the build. Only touch this when adding or removing CSS files.

## What Not To Edit

- `build/AstrlGlass.css`: generated by the build script.
- `dist/AstrlGlass.theme.css`: generated single-file theme.
- `node_modules/`: installed packages.

If you edit generated files by hand, your changes can disappear the next time the watcher rebuilds.

The project name and filenames are AstrlGlass now.

## DMs Icon Bounce

The easy bounce settings are in `themes/AstrlGlass.theme.css`:

```css
--dms-icon-svg-transition: 0.95s;
--dms-icon-bounce-up: -8px;
--dms-icon-bounce-down: 3px;
--dms-icon-bounce-settle: -2px;
```

Negative numbers move the icon up. Positive numbers move it down. Bigger numbers make the bounce more dramatic.

## Live Testing

Run the watcher:

```powershell
pnpm dev
```

Then enable `AstrlGlass-dev.theme.css` in Discord. If one edit does not visually update, toggle the theme off/on or press `Ctrl+R` in Discord.

## Scripts

- `scripts/build.js`: combines `src/*.css` into generated theme files.
- `scripts/dev.js`: watches your files and writes the live Discord theme file from `.env`.
- `scripts/theme.config.js`: controls source file order and generated file paths.

## Publishing

Because the repo is public, other people can install the online `themes/AstrlGlass.theme.css` import.

The generated `dist/AstrlGlass.theme.css` file can still be installed manually because it contains everything in one file.
